# 💰 Planej.ai

**Planej.ai** é uma aplicação de planejamento financeiro pessoal que utiliza Inteligência Artificial para ajudar os usuários a organizarem suas finanças de forma inteligente e automatizada.

## 📌 O que o projeto faz
O Planej.ai simplifica a gestão financeira. O usuário insere suas receitas e despesas, e a aplicação (com o auxílio de IA) categoriza os gastos, analisa o padrão de consumo e 
fornece dicas personalizadas para economizar e investir melhor. O objetivo é tirar o peso do planejamento manual e oferecer *insights* práticos para a saúde financeira.

## 🚀 Como executar a aplicação

Para rodar o projeto localmente, siga os passos abaixo:

1. Clone este repositório:
```bash
      git clone https://github.com/Samw1s3dev/Planej.ai.git

```

2. Acesse a pasta do projeto:
```bash
   cd Planej.ai

```


3. Instale as dependências:
```bash
   npm install

```


4. Configure as variáveis de ambiente:
* Crie um arquivo `.env.local` na raiz do projeto.
* Adicione suas chaves (ex: chave da API de IA): `VITE_GEMINI_API_KEY=sua_chave_aqui`


5. Inicie o servidor:
```bash
   npm run dev

```



Acesse `http://localhost:5173` no seu navegador.

## 🛠️ Quais tecnologias foram usadas

* Front-end: React, Next.js, Tailwind CSS
* Back-end:  Node.js
* Inteligência Artificial: Integração com a API do Google Gemini 


## 🧪 Como testar o fluxo principal

Para validar o funcionamento da aplicação, siga este roteiro:

1. Adicione uma nova receita, e responda aos cards seguintes.
2. Clique no botão "Gerar simulação", e aguarde a resposta da IA.
